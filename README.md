# üõí Dynamic Retail Dashboard (Walmart Sales Analysis)

This repository contains a **Dynamic Excel Retail Dashboard** created for analyzing Walmart sales performance across different segments, categories, and countries.  
The dashboard provides **interactive visual insights** into sales, profit, and quantity metrics using Excel‚Äôs data visualization tools.

---

## üìò Project Overview
The **Dynamic Retail Dashboard** is designed to help retail managers and analysts monitor business performance in real time.  
It uses **Microsoft Excel** features like Pivot Tables, Charts, Slicers, and Data Modeling to create a visually appealing, data-driven interface ‚Äî all without coding.

---

## üéØ Objectives
- Analyze **sales, profit, and quantity** metrics across various product categories and regions.  
- Identify **top-performing and underperforming subcategories**.  
- Understand **country-wise profit distribution** using a global map.  
- Track **profitability trends** and order quantities over time.  
- Provide dynamic filtering by **segment, category, region, ship mode, and customer ID**.

## üìä Dashboard Components
| Section | Description |
|----------|-------------|
| **KPI Cards** | Show total sales, total profit, quantity sold, order count, and profitability percentage |
| **Category-wise Profit** | Pie chart comparing profit across product categories |
| **Segment-wise Profit** | Donut chart visualizing profit by consumer segments |
| **Total Profit by Country** | World map showing profit distribution by geography |
| **Top 5 & Bottom 5 Subcategories** | Compare best and least profitable subcategories |
| **Sales vs Profit** | Scatter plot showing relationship between sales volume and profit margin |
| **Filters / Slicers** | Interactive filters for Region, Segment, Category, Ship Mode, and Customer ID |
